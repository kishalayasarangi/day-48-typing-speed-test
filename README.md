# Day 48 — Typing Speed Test

**120 Days of Code** | NIT Rourkela | Mechanical Engineering

## What this is
A typing speed test that measures WPM, accuracy and
consistency with multiple difficulty levels.

## How to view
Open `index.html` in your browser.

## Features
- Measures WPM and accuracy in real time
- 3 difficulty levels (Easy, Medium, Hard)
- Live error highlighting
- Results history saved to localStorage
- Personal best tracking
